AULA_5: Métodos Avançados da biblioteca time

    O módulo time em Python fornece uma variedade de funções para trabalhar com tempo.
        strftime():
            "converte um objeto de tempo struct para uma string de acordo com um formato específico."
        strptime():
            "analisa uma string representando um horário de acordo com um formato. O retorno é um objeto `struct_time`."
        gmtime():
            "converte um tempo expresso em segundos desde a epoch em um objeto struct_time em UTC."
        mktime():
            "converte um objeto `struct_time` em segundos desde a epoch. Este é o inverso da função `localtime()`."
    
    "Para obter o tempo formatado em português, podemos usar o módulo locale do Python."
