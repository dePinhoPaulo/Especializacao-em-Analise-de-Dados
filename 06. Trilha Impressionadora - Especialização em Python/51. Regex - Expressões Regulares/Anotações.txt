AULA_2: Caracteres Especiais no Regex

    re.compile('padrao_regex') -> compilar um padrão regex
    re.search(padrao_compilado, texto) -> procura uma ocorrência do padrão no texto (re.match só procura na 1ª linha do texto)
    re.findall(padrao_compilado, texto) -> encontra todas as ocorrencias do padrão em um texto - armazena em uma lista
    re.finditer(padrao_compilado, texto) -> encontra todas as ocorrencias e armazena em um iterador

    Contar aparições de um caractere em uma string:
        padrao = re.compile(r"\$")
        resultado = re.findall(padrao, texto)
        print(len(resultado))


AULA_3: O bom uso de Regex - Encontrando padrões

    Sempre utiliza Raw string:
        padrao = re.compile(r'\d')

    buscando todos os numeros:
        padrao = re.compile(r'\d')
        resultado = re.findall(padrao, texto)
        print(resultado)
    
    buscando qualquer numero de 1 ou mais digitos:
        padrao = re.compile(r'\d+')
    
    buscando qualquer numero de 1 ou mais digitos com virgula no meio:
        padrao = re.compile(r'\d+,\d+')


AULA_4: Como analisar a resposta que o regex vai encontrar

    Buscar todas strings de uma uo mais letras:
        novo_texto = 'Nome: Paulo Idade: 24 Salario: 50000'
        padrao = re.compile(r'\D+')
        resultado = re.findall(padrao, novo_texto)
        print(resultado)

    Buscar strings que estão antes do dois pontos:
        padrao = re.compile(r'(\w+):')
        "parentese faz isolar a busca (como na matematica que reslove primeiro)"

    Buscar Todos os numeros (1 digito ou mais, com nenhuma virgula ou uma, com zero ou mais digitos)
        padrao = re.compile(r'\d+,?\d*')


AULA_5: Outros exemplos para treinar

    Função para retornar 1° item:
        re.search()

    Extraindo ml do 1° item: 
        padrao = re.compile(r'\(.+\)') ou padrao = re.compile(r'\(\d+\s\w+\)')
        resultado = re.search(padrao, texto)
        print(resultado)

    Extraindo 1° Preço:
        padrao = re.compile(r'R\$\d+,?\d+')
        resultado = re.search(padrao, texto)
        print(resultado)
        print(resultado.group(0)) # Traz extamente a string requisitada (não em formato de objeto como padrão)


AULA_6: Sequências de Caracteres

    Busca todo preços, mas com apens 2 digitos após a virgula:
        padrao = re.compile(r'R\$\d+,?\d{2}')
        resultado = re.findall(padrao, texto)
        print(resultado)
        "Chaves limitam a quantidade de caracteres"
    
    Busca todo preços, mas que comecem com digitos de 0 a 5:
        padrao = re.compile(r'R\$[0-5]+,?\d{2}')
        resultado = re.findall(padrao, texto)
        print(resultado)
        "Colchetes filtra os caracteres, o mesmo vale para letras [a - f]"


AULA_7: Diferentes formas de resposta no Regex

    Buscando todas quantidade de itens, sem retornar os parenteses:
        padrao = re.compile(r'\((.+)\)')
        resultado = re.findall(padrao, texto)
        print(resultado)

AULA_9: Outros operadores de Regex

    ^ – buscar elementos no início da string;
        padrao = re.compile(r'^http')

    $ – buscar elementos no final da string;
        padrao = re.compile(r'.com/?$')

    [^] - diferente de um caractere especificado logo após o ^
        padrao = re.compile(r'http[a-z:/]+\.[^g].+')
